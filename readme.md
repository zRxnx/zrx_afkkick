# zRxnx's Anti AFK System

A simple resource to stop afk players

<b>Features:</b>
- Kick people after x time
- Checks before execution
- Fancy discord log
- 0.0 ms while in use

## Requirements
- none

## Installation
- Download script
- Put script in your `resources` directory
- Add `ensure zrx_afkkick` in your `server.cfg`
